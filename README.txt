5. Write recursive functions for the following tasks:
   (i) reverse the order of elements of the SinglyLinkedList class,
   (ii) print the elements of the SinglyLinkedList class, in reverse order.
   Write the code and annotate it with proper loop invariants.
   Running time: O(n).

How to Compile:
1. Unzip contents of lxj171130.zip into a folder named "lxj171130".
2. Run the command `javac lxj171130/ReverseSinglyLinkedList.java` to compile.
3. Run the command `java lxj171130.ReverseSinglyLinkedList` to execute the program.
Note: option 3 for reversing the Singly Linked List
